# Dashboard_Backbone_2026
Dashboard interactivo con la base de datos de Backbone actualizada ('2026.01.01 - BB Social Impact Data S2-2025'). Se agrega la pestaña de precio promedio al análisis.

Bases de datos utilizadas:
  - Datos de clínicas de Backbone: 2026.01.01 - BB Social Impact Data S2-2025.xlsx
  - Censo de Población y Vivienda (CPV) (INEGI, 2020): https://www.inegi.org.mx/programas/ccpv/2020/#datos_abiertos
  - Grado de Rezago Social a Nivel AGEB urbana (Coneval, 2020): https://www.coneval.org.mx/Medicion/IRS/Paginas/Rezago_social_AGEB_2020.aspx
  - NSE Desagregado por Manzana, AGEB, Localidad y Municipio (AMAI, 2024): https://www.amai.org/NSE/index.php?queVeo=NSEDES
